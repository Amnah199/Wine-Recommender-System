# Wines.MS

## Description

Wine MS is recommendation system consists of large wine database with detail information along with recommendation engine for generating the most similar type according to your taste. Users are supposed to insert their preferences for wine taste and origin, which can be further modified with spider diagram for making more precise taste range. Engine will consider you choice and recommend you to most suitable wine, which you can buy in wine shops all around the MÃ¼nster.
